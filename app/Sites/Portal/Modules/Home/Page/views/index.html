A PHP modern, scalable and light MVC Framework.

namespace App\Sites\Portal\Modules\Home\Page;

use Roducks\Page\Page;


class Home extends Page {
  public function index()
  {
      // Pass variables to view <KEY>, <VALUE>
      $this->view->data('title', "Home");

      // Load View
      $this->view->load('index');


      return $this->view->output();
  }
}

It works with a MySQL database that includes its own ORM for SQL transactions as many other features and libraries to make a great and secure Web Application.
